## 1. 프로젝트 소개

MatchCrew(매치크루)는 팀원들을 소개하고, 간단한 카드 매치 플레이를 통해 스테이지를 클리어하는 게임입니다.
단계별로 구성된 스테이지를 하나씩 진행하며, 팀원들의 개성과 매력을 담은 카드를 확인해보세요!
